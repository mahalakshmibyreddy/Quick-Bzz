COLLABORATION TOOLS

Collaboration tools allow students, instructors, and teaching assistants to exchange resources in a number of different ways, depending on what is needed for a particular task. Tools include document and resource sharing and collaboration, as well as web conferencing.

Team Tasks

Proofhub gives users a range of features, which help them bolster collaboration among their team. These features include group chats, discussions, workflows, project reports, and more.

Zoom provide an excellent voice and video service for virtual communication, but it also includes team chat, whiteboarding, email and calendar integrations. Users can also create team workspaces to share.

Figma is a collaborative platform for designers which allows multiple teams to collaborate.Teams can edit designs together in real-time,Integrations with Trello, Flinto, Slack, Dropbox, and Jira

File Sharing

File sharing is the public or private sharing of files or folders on a computer connected to a network. Files can easily be shared outside a network via removable media, but the term file sharing almost always refers to sharing files on a network. File sharing allows several people to use the same file data.

File Sharing & Storage Tools

Citrix Systems, Inc. is an American multinational cloud computing and virtualization technology company that provides server, application and desktop virtualization, networking, software as a service (SaaS), and cloud computing technologies.

iCloud is a free service, and comes with 5 GB of cloud storage. Users can subscribe to iCloud+ for additional storage up to 2 TB (or 4 TB for users of Apple One Premier plan who also buy an additional 2 TB of storage).

Dropbox brings files together in one central place by creating a special folder on the user's computer.The contents of these folders are synchronized to Dropbox's servers and to other computers and devices where the user has installed Dropbox.

WeTransfer was founded in 2009 by Rinke Visser, Bas Beerens and Ronald Hans (Nalden) in Amsterdam, the Netherlands.It was created to enable the sharing of large files (up to 2GB) free of charge.

Messaging

Instant messaging programs enable you to exchange information or messages instantly in real-time. Businesses use such programs for effective communication within and across the teams. Instant software is an online application that enables users to collaborate with their members and make informed decisions.

WeChat had over 889 million monthly active users by 2016, and as of 2019 WeChat's monthly active users had risen to an estimate of one billion. As of January 2022, it was reported that WeChat has more than 1.2 billion users.

Slack is a cloud-based team communication platform developed by Slack Technologies, which has been owned by Salesforce since 2020. Slack uses a freemium model. Slack is primarily offered as a business-to-business service.

Chatwork is an all-round instant messaging chat app that is designed to help global teams communicate, collaborate and increase productivity. The platform includes secure messaging, video chat, task management and file sharing functionality.

Viber is another messaging app that is growing globally with more than1 billion users worldwide. It is extremely popular in Eastern Europe, but unlike Facebook and WhatsApp, it is not that strong within the business sphere.
